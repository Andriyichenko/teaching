\textbf{問2}\qquad $3a + 1$ が $a^2 + 6$ の約数となるような自然数 $a$ を求めよう。

\vspace{2em}

$3a + 1 = b$ とする。このとき

\vspace{1em}

\begin{center}
$a^2 + 6 = \dfrac{b^2 - \ab{\textsf{\color{red}F}}\ b + \ab{\textsf{\color{red}GH}}}{\ab{\textsf{\color{red}I}}}$ \hspace{2em} $\cdots\cdots$ \maru{1}
\end{center}

\vspace{1em}

である。また、$b$ は $a^2 + 6$ の約数であるから、$a^2 + 6$ はある自然数 $c$ を用いて

\vspace{1em}

\begin{center}
$a^2 + 6 = bc$ \hspace{2em} $\cdots\cdots$ \maru{2}
\end{center}

\vspace{1em}
と表される。\maru{1}、\maru{2} から

\vspace{1em}

\begin{center}
$b\left(\ab{\textsf{\color{red}J}}\ c - b + \ab{\textsf{\color{red}K}}\right) = \ab{\textsf{\color{red}LM}}$
\end{center}

\vspace{1em}

を得る。したがって、$b$ は \abb{\textsf{\color{red}LM}} の約数である。この中で、$a$ が自然数となるのは $b =$ \ab{\textsf{\color{red}NO}}

\vspace{1em}

である。したがって、$a = \ab{\textsf{\color{red}PQ}}$ である。

$n$ は 2 \ruby{桁}{けた}の自然数であり、$n^3$ を 78 で割ったときの余りは $n$ であるという。このような $n$
\noindent
の個数と、このような $n$ のうち素数であるものを求めよう。

\vspace{2em}

条件より、$n^3$ を 78 で割ったときの商を $p$ とすると

\vspace{1em}

\hspace{3em} $n^3 = \ab{\textsf{\color{red}AB}}\ p + n$ \hspace{2em} $\left(0 < n \leq \ab{\textsf{\color{red}CD}}\right)$

\vspace{1em}
\noindent
と表せる。これを変形して 

\vspace{1em}

\hspace{3em} $n(n - 1)(n + 1) = \abb{\textsf{\color{red}AB}}\ p$

\vspace{1em}
\noindent
を得る。

\vspace{1em}
ここで、$n - 1, n$ のどちらか一方は \ab{\textsf{\color{red}E}} の倍数、$n - 1, n, n + 1$ のうち 1 つは \ab{\textsf{\color{red}F}} の倍数であり、\\

\noindent
\abb{\textsf{\color{red}E}} と \abb{\textsf{\color{red}F}} は互いに素であるから、$n(n - 1)(n + 1)$ は \ab{\textsf{\color{red}G}} の倍数である。\\

\noindent
ただし、$1 < \abb{\textsf{\color{red}E}} < \abb{\textsf{\color{red}F}} < \abb{\textsf{\color{red}G}}$ とする。よって、$n - 1, n, n + 1$ のいずれか
\vspace{1em}
1 つが \ab{\textsf{\color{red}HI}} の倍数となる場合を考えればよい。

\vspace{1em}

いま、$n \leq \abb{\textsf{\color{red}CD}}$ であるから、$n - 1$ が \abb{\textsf{\color{red}HI}} の倍数である $n$ の個数は \ab{\textsf{\color{red}J}}、$n$ が \abb{\textsf{\color{red}HI}} の倍数である $n$ の個数\\

\noindent
は \ab{\textsf{\color{red}K}}、$n + 1$ が \abb{\textsf{\color{red}HI}} の倍数である $n$ の個数は \ab{\textsf{\color{red}L}}である。

\vspace{2em}

よって、求める $n$ の個数は \ab{\textsf{\color{red}MN}} であり、このうち、素数である $n$ は小さい順に \ab{\textsf{\color{red}OP}} 、\ab{\textsf{\color{red}QR}} 、\ab{\textsf{\color{red}ST}} である。

2 つの実数 $x, y$ が方程式

\vspace{1em}

$$3x^2 + 2xy + 3y^2 = 50 \hspace{3em} \cdots\cdots \maru{1}$$ 

\vspace{1em}

\noindent
を満たしている。このとき、$x + y$ および $xy$ がとる値の範囲を求めよう。(結果は既約分数で表せよ。)

\vspace{2em}

まず

\vspace{1em}

\hspace{3em} $x + y = a$ \hspace{3em} $\cdots\cdots$ \maru{2}

\vspace{1em}

\noindent
とおく。\maru{1}、\maru{2} より $y$ を消去して、$x$ の 2 次方程式

\vspace{1em}

$$ \ab{\textsf{\color{red}A}}\ x^2 -\ \ab{\textsf{\color{red}B}}\ ax +\ \ab{\textsf{\color{red}C}}\ a^2 - 50 = 0$$

\vspace{1em}

\noindent
を得る。$x$ は実数であるから

\vspace{1em}

$$\ab{\textsf{\color{red}DE}} \leq a \leq \ab{\textsf{\color{red}F}} \hspace{3em} \cdots\cdots \maru{3}$$

\vspace{1em}

\noindent
である。

\vspace{1em}

さらに

\vspace{1em}


$$xy = b \hspace{8em} \cdots\cdots \maru{4}$$

\vspace{1em}

\noindent
とおくと、\maru{1}、\maru{2}、\maru{4} より

\vspace{1em}

$$b = \frac{\ab{\textsf{\color{red}G}}\ a^2- \ab{\textsf{\color{red}IJ}}}{\ab{\textsf{\color{red}H}}}  \hspace{3em} \cdots\cdots \maru{5}$$

\vspace{1em}

\noindent
を得る。よって、\maru{3}、\maru{5} より

\vspace{1em}

$$\dfrac{\ab{\textsf{\color{red}KLM}}}{\ab{\textsf{\color{red}N}}} \leq b \leq \dfrac{\ab{\textsf{\color{red}OP}}}{\ab{\textsf{\color{red}Q}}}$$

\vspace{1em}

\noindent
となる
